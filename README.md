# 🏏 ScoreSync

**ScoreSync** is a full-stack live cricket score tracking application. It allows users to create cricket teams with custom names and logos, add players, schedule matches between teams, and update live scores. Inspired by platforms like Cricbuzz, ScoreSync provides a seamless experience for both hosts (score managers) and viewers.

---

## 🚀 Features

- **Team Management**
  - Create teams with custom names and logos
  - Add and manage players within each team

- **Match Management**
  - Host can schedule matches between created teams
  - Assign players to matches
  - Live scoreboard updates by the host

- **Live Score Tracking**
  - Real-time score updates
  - Scoreboard visible to all users

- **Authentication**
  - Sign up and login functionality
  - Role-based access: Host and Viewer
